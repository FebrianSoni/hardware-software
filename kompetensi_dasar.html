📖 Kompetensi Dasar (KD)

Kompetensi Dasar Mata Pelajaran Informatika

1. Pemahaman tentang Hardware

3.1 Memahami berbagai komponen hardware pada komputer:

  • Mengidentifikasi dan menjelaskan fungsi masing-masing komponen hardware seperti CPU, RAM, hard disk, motherboard, kartu grafis, dan perangkat input/output (keyboard, mouse, monitor, dll).
  • Menjelaskan perbedaan antara hardware internal dan eksternal.
  • Memahami cara kerja komponen-komponen hardware dalam memproses data.

3.2 Memahami konsep dasar sistem komputer:

  • Menjelaskan hubungan antara hardware, software, dan pengguna.
  • Memahami bagaimana data diproses dalam sistem komputer.
  • Menjelaskan proses booting komputer.

4.1 Mempraktikkan perakitan dan perawatan hardware dasar:

  • Melakukan instalasi komponen hardware dasar seperti RAM, kartu grafis, atau hard disk (opsional).
  • Melakukan perawatan dasar hardware seperti membersihkan debu atau mengganti pasta termal (opsional).

2. Pemahaman tentang Software

3.3 Memahami berbagai jenis software:

  • Menjelaskan perbedaan antara sistem operasi (OS) dan aplikasi.
  • Menyebutkan contoh-contoh sistem operasi (Windows, macOS, Linux, dll.) dan aplikasi (Microsoft Office, browser web, game, dll.).
  • Memahami fungsi masing-masing jenis software dalam menjalankan tugas tertentu.

3.4 Memahami cara kerja software dalam berinteraksi dengan hardware:

  • Menjelaskan peran sistem operasi dalam mengelola hardware dan sumber daya komputer.
  • Memahami konsep driver dan bagaimana mereka memungkinkan software berkomunikasi dengan hardware.

4.2 Menggunakan berbagai aplikasi software untuk menyelesaikan tugas-tugas tertentu:

  • Menggunakan aplikasi pengolah kata, angka, atau presentasi untuk membuat dokumen, menghitung, atau menyajikan informasi.
  • Menggunakan aplikasi browser web untuk mencari informasi atau mengakses layanan online.
  • Menggunakan aplikasi lain yang relevan dengan kebutuhan pembelajaran seperti desain grafis atau pemrograman.

3. Interaksi Hardware dan Software

3.5 Memahami interaksi antara hardware dan software dalam menjalankan aplikasi:

  • Menjelaskan bagaimana data diproses dan ditampilkan pada monitor.
  • Menjelaskan bagaimana input dari pengguna (seperti keyboard) diproses dan ditampilkan.

4.3 Memecahkan masalah sederhana yang terkait dengan hardware dan software:

  • Mendiagnosis masalah umum seperti komputer tidak mau menyala, aplikasi tidak merespons, atau masalah koneksi internet.
  • Mencari solusi untuk masalah tersebut dengan bantuan informasi atau konsultasi dengan guru/ahli.
