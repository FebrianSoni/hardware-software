🧩 Pengertian Hardware & Software

Hardware adalah komponen pada sebuah sistem komputer yang dapat dilihat dan disentuh. Contohnya seperti CPU, Monitor, Keyboard, Mouse, dll.

Software adalah program atau instruksi yang mengatur kerja hardware. Contohnya seperti Windows, Microsoft Word, Photoshop, dll.

Hardware dan software bekerja saling bergantung satu sama lain untuk membuat komputer dapat berfungsi secara optimal.

🔧 Fungsi Hardware dan Software

🛠️ Fungsi Hardware

Hardware melakukan pekerjaan fisik, seperti memproses data (CPU), menyimpan data sementara (RAM), memberi input (keyboard), dan output (monitor/printer).

💡 Fungsi Software

Software mengatur penggunaan hardware. Sistem operasi mengelola memori, CPU, dan perangkat input/output. Aplikasi memungkinkan pengguna menjalankan tugas-tugas tertentu.
