Hardware adalah komponen fisik dari komputer, seperti CPU, monitor, keyboard, dan mouse.

1. CPU (Central Processing Unit)

CPU adalah otak dari komputer. Bertugas menjalankan instruksi dan mengendalikan operasi komputer.

Funggsi CPU

  • Pemrosesan data
  • Pengendalian operasi
  • Eksekusi instruksi
  • Komunikasi dengan perangkat lain
Klik untuk lihat contoh CPU

2. RAM (Random Access Memory)

RAM menyimpan data sementara yang dibutuhkan saat aplikasi berjalan. Membantu kinerja komputer agar tetap cepat dan responsif.

Funggsi RAM

  • Membantu CPU memproses data dengan cepat
  • Menyimpan data sementara
  • Meningkatkan performa saat multitasking
Klik untuk lihat contoh RAM

3. Monitor

perangkat keras (hardware) pada komputer yang berfungsi untuk menampilkan visual atau gambar, teks, video, dan grafik yang dihasilkan oleh unit pemroses pusat (CPU). Monitor adalah bagian penting dari sistem komputer yang memungkinkan pengguna berinteraksi dan melihat hasil pengolahan data.

Funggsi Monitor

  • Menampilkan Output Komputer
  • Menyediakan Antarmuka Pengguna
  • Media Hiburan
  • Pengawasan dan Pengendalian
Klik untuk lihat contoh monitor

4. Keyboard

Keyboard adalah perangkat input utama yang digunakan untuk memasukkan huruf, angka, dan perintah ke dalam komputer.

Fungsi Keyboard

  • Memasukkan data dan instruksi
  • Mengontrol perangkat lunak melalui kombinasi tombol
Klik untuk lihat contoh keyboard

5. Mouse

Mouse adalah perangkat input yang digunakan untuk mengarahkan pointer dan memilih objek di layar komputer.

Fungsi Mouse

  • Mengontrol pergerakan kursor di layar
  • Memilih, membuka, dan menyeret objek
  • Memberikan input melalui klik kanan, kiri, dan scroll
Klik untuk lihat contoh mouse

6. Hardisk

Hardisk adalah perangkat penyimpanan data permanen yang digunakan pada komputer.

Fungsi Hardisk

  • Menyimpan sistem operasi, aplikasi, dan file data
  • Mengelola partisi dan penyimpanan komputer
  • Menyimpan data dalam jangka panjang meski komputer dimatikan
Klik untuk lihat contoh hardisk

7. Printer

Printer adalah perangkat output yang mencetak dokumen atau gambar dari komputer ke media kertas.

Fungsi Printer

  • Mencetak teks atau gambar dari file digital
  • Memberikan keluaran fisik berupa dokumen
  • Mendukung pencetakan berwarna maupun hitam-putih
Klik untuk lihat contoh printer

8. Scanner

Scanner adalah perangkat input yang digunakan untuk menyalin dokumen fisik ke format digital.

Fungsi Scanner

  • Memindai dokumen dan gambar menjadi file komputer
  • Mengubah dokumen kertas ke PDF atau gambar
  • Mendukung digitalisasi dokumen fisik
Klik untuk lihat contoh scanner

9. Joystick

Joystick adalah perangkat input yang digunakan untuk mengontrol gerakan dalam game atau simulasi.

Fungsi Joystick

  • Mengendalikan arah dalam game atau simulasi
  • Memberikan input analog untuk gerakan halus
  • Mempermudah kontrol dalam aplikasi interaktif
Klik untuk lihat contoh joystick

10. Touchpad

Touchpad adalah perangkat input berupa permukaan datar yang digunakan sebagai pengganti mouse, umumnya pada laptop.

Fungsi Touchpad

  • Mengontrol kursor dengan sentuhan jari
  • Menjalankan klik, geser, dan gesture
  • Menghemat ruang karena terintegrasi di laptop
Klik untuk lihat contoh touchpad

11. Speaker

Speaker adalah perangkat output yang digunakan untuk mengeluarkan suara dari komputer.

Fungsi Speaker

  • Memutar audio dari musik, video, dan aplikasi
  • Menyampaikan notifikasi dan sistem suara
  • Meningkatkan pengalaman multimedia pengguna
Klik untuk lihat contoh speaker

12. Motherboard

Motherboard adalah papan sirkuit utama tempat semua komponen hardware komputer terhubung.

Fungsi Motherboard

  • Menghubungkan CPU, RAM, dan semua perangkat input/output
  • Menyediakan jalur komunikasi antar komponen
  • Mengelola daya dan sinyal antar hardware
Klik untuk lihat contoh motherboard

13. VGA Card (Video Graphics Adapter)

VGA Card adalah perangkat keras yang bertanggung jawab mengolah data visual menjadi tampilan grafis di layar monitor.

Fungsi VGA Card

  • Mengubah sinyal digital menjadi tampilan grafis
  • Mempercepat kinerja grafis, terutama game dan editing
  • Menampilkan visual berkualitas tinggi di monitor
Klik untuk lihat contoh VGA Card
