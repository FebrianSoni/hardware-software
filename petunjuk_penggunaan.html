Cara Menggunakan Media Ini

  1. Baca materi Hardware dan Software melalui menu di atas.
  2. Gunakan tombol interaktif untuk melihat gambar dan contoh perangkat.
  3. Setelah memahami materi, klik menu Soal Latihan untuk menguji pemahaman.
  4. Lihat menu Kompetensi Dasar untuk mengetahui tujuan pembelajaran.
  5. Gunakan perangkat dekstop (komputer) dan mobile (smartphone) dengan browser terbaru agar tampilan optimal.
